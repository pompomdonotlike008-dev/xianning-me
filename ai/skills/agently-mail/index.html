agently-mail：自然语言操作 QQ 邮箱

2026-06-27Skill

一句话：用自然语言让 Claude Code 帮你发邮件、查收件箱、回复邮件、下载附件。基于腾讯 Agent 原生邮箱方案。

能力

  • ✉️ 发送邮件——"帮我给张三发一封邮件，内容是……"
  • 📥 搜索邮件——"找一下上周张三发的那封关于合同的邮件"
  • ↩️ 回复/转发——"回复李四的邮件，说收到，明天处理"
  • 📎 下载附件——"把王五发来的 PDF 下载下来"
  • 📊 管理收件箱——"把未读的营销邮件归档"

技术架构

Claude Code (自然语言指令)
    │
    ▼
agently-mail Skill
    │
    ▼
agently-cli (OAuth 授权 + QQ 邮箱 API)
    │
    ▼
QQ 邮箱服务器

核心是 agently-cli——一个腾讯提供的命令行邮件工具，通过 OAuth 授权访问 QQ 邮箱。Skill 负责把 Claude Code 的自然语言指令翻译成 agently-cli 的命令行参数。

使用场景

场景 1：批量处理

"把所有标记为重要但超过一周没回的邮件列出来，草拟回复。"

场景 2：附件提取

"找出上个月所有带 xlsx 附件的邮件，下载附件，汇总数据。"

场景 3：邮件总结

"总结一下今天的邮件，按重要性排序，列出需要我回复的。"

安全设计

  • 🔐 OAuth 授权，不需要提供邮箱密码
  • 🔐 权限可随时在 QQ 邮箱设置中撤销
  • 🔐 所有操作通过 agently-cli 代理，Skill 不直接接触邮件数据
  • 🔐 发送邮件前会展示预览，用户确认后才执行

理念：Agent 原生邮箱——不是「帮你看邮件」，而是「你的 Agent 有自己的邮箱，可以独立完成邮件任务」。

📥 下载安装

把 SKILL.md 放到 ~/.claude/skills/agently-mail/SKILL.md，重启 Claude Code 即可使用。
