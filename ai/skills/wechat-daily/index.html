wechat-daily：公众号日报自动生成与草稿推送

2026-06-27Skill

一句话：每天自动搜集新能源和 AI 动态，排版成日报格式，直接创建公众号草稿。你只需要手机上点一下发布。

工作流

1. 信息搜集
    ├─ 搜索新能源政策动态（国家能源局、各省发改委）
    └─ 搜索 AI 工具/模型动态（GitHub、技术博客）
2. 编译排版
    ├─ 格式化标题 + 摘要 + 链接
    ├─ 插入公众号固定模板
    └─ 添加免责声明和关注引导
3. 注入草稿
    ├─ 通过 CDP 操作微信公众平台后台
    ├─ 读取当前模板 HTML
    └─ 创建草稿，填入 HTML
4. 完成 → 用户在手机端点「发布」

关键细节

模板 HTML 结构会变

微信后台的草稿模板 HTML 结构不是固定的，版本更新时可能改变。每次注入前需要重新读取模板，动态匹配当前的 DOM 结构。

自动化 ≠ 全自动

最终发布权始终在用户手里。Skill 只做采集 + 排版 + 草稿，不做自动发布。这既是安全边界，也是质量把控。

CDP 驱动的浏览器操作

登录态通过 Chrome DevTools Protocol 维持。需要在本地运行一个 Chrome 实例，Skill 通过 CDP 连接到微信后台进行操作。
